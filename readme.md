```Web Developer Assignment```

This project is my implementation of a Figma design using HTML and CSS. The goal was to build a fully responsive and accessible web page that matches the provided design as closely as possible.

Key Features

- Responsive layout (desktop, tablet, and mobile)

- Breakpoints on 1325px, 1050px.

- Left Sidebar list items have active style, if we click another list item active style applies on that particular item.

- Left Sidebar hidden on smaller screen size devices, Visible after clicking on header menu button.

- Semantic HTML for better accessibility
